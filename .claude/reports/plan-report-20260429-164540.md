# 並列計算モジュール開発

## 概要

`example/src/` に計算モジュールを並列 TDD 開発する。
2つのタスクは互いに独立しており、並列実行する。

## 実装対象

### addition.py
- 関数名: `add(a, b)`
- 仕様: 2つの数値（int / float）を受け取り、足し算の結果を返す
- ファイルパス: `example/src/addition.py`
- テストパス: `example/tests/test_addition.py`

### multiplication.py
- 関数名: `multiply(a, b)`
- 仕様: 2つの数値（int / float）を受け取り、掛け算の結果を返す
- ファイルパス: `example/src/multiplication.py`
- テストパス: `example/tests/test_multiplication.py`

## テスト要件

各関数に対して pytest テストを作成すること:
- 正の整数同士の演算
- 0 との演算
- 負の数を含む演算
- float 同士の演算

## 実装言語・ツール

- Python 3.11
- pytest（テストフレームワーク）
- 外部ライブラリなし（標準ライブラリのみ）
