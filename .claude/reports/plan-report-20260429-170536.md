# 並列計算モジュール開発

## 概要

`example/src/` に計算モジュールを並列 TDD 開発する。

## 実装対象

### addition.py
- 関数名: `add(a, b)`
- 仕様: 2つの数値を受け取り、足し算の結果を返す
- ファイルパス: `example/src/addition.py`
- テストパス: `example/tests/test_addition.py`

### multiplication.py
- 関数名: `multiply(a, b)`
- 仕様: 2つの数値を受け取り、掛け算の結果を返す
- ファイルパス: `example/src/multiplication.py`
- テストパス: `example/tests/test_multiplication.py`

## テスト要件

- 正の整数・0・負の数・float での演算テスト

## 実装言語・ツール

- Python 3.11 / pytest / 外部ライブラリなし
